# OdontoartPix

Base inicial do sistema web de análise de mensalidades.

## Variáveis

Copie `.env.example` para `.env.local` e preencha as credenciais do Supabase e da API externa.

## Estrutura inicial

- Next.js App Router
- TypeScript
- Supabase
- Zod
- Função isolada de análise financeira
- Rota server-side para consultar a API externa sem expor token

## Testes

```bash
npm run test
```
